Homework/Binary Adaptation: Create a python function that will convert a decimal number 1-255 to binary using mathematical operations and powers of 2. Challenge: add frontend with javascript or html.

def convert(number):
    if number > 1:
        convert(number//2)
    print(number%2, end = "")
